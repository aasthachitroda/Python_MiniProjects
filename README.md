# Python Mini Projects

A collection of fun and beginner-friendly Python projects built to practice logic and programming skills.

## Projects

- **Higher Lower Game**  
  A simple comparison game where the player guesses which number is higher.

- **Bulls and Cows Game**  
  A number guessing game where players try to guess a secret number using clues.

- **Currency Converter**  
  Converts values between different currencies using exchange rates.

- **WhatsApp Birthday Bot**  
  Sends automated birthday wishes on WhatsApp using Python scripting.

- **Word Guessing Game**  
  A text-based game where the player guesses a hidden word, one letter at a time.
